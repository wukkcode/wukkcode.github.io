    • Windows 11抢先体验【现代UI设计】

    2021年6月29日我回到了家，作为微软粉，回到家的第一件事就是【更新Windows】

    Windows11本周会在dev频道开始测试，正式稳定版据说是在10月份左右公布：微软意外透露 Win11 正式版推出时间
    我们可以打开【设置】——>【Windows更新】——>【Dev渠道】

    当然，在开始更新前，我们可以先检测一下自己的配置是否符合要求，为此微软推出了一个电脑健康状况检查工具，但是微软承认其 Win11 升级电脑健康状况检测工具存在 bug，现已下架。

    但是不用担心，有开发者连夜开发了一款比较好的检测工具：WhyNotWin11

    这里比较关键的是【CPU版本】和【TPM安全检测模块】

    当然这些检测指标不是硬性得，你可以在预览窗口看一下自己得配置是否满足【最低硬件要求】
    大量 Win10 用户无法推送升级 Win11 Dev 预览版：不满足最低硬件要求，频道选择受限
    如果不满足最低硬件要求，一般是不会收到最新得Windows 11版本更新的。

    选择好之后就可以重启电脑，如果一切顺利，更新窗口就会收到【Window11更新提示】
    然后就是漫长的下载更新和安装过程，这个时候给计算机通上电源，倒一杯茶，然后静静等待。

    首先是【开机logo】的变化

    倾斜logo变成正面logo

    【Windows聚焦】对比性也更强了

    开机之后就是一个全新的Windows了

    居中的任务栏，统一的Fluent图标，浮动的开始菜单，还有圆角窗口

    新加了一个【小组件】功能

    现阶段只是一些可有可无的组件功能

    【文件资源管理器】也进行了重新设计

    UI材质并没有使用原来的毛玻璃半透明，而采用是一种新的UI材质：Mica云母材质
    因为毛玻璃材质对性能要求比较高

    还有万年不变的【右键菜单】也更新了

    【设置】界面采用新的UI和交互逻辑

    也自带了几款系统主题，但现阶段只是壁纸和颜色上的不同。

    【微软商店】也更新了，使用【原生UWP】开发，性能更好。

    【输入法】UI也更新了：圆角，上下边距更小了，更加静止了。

    除此之外，还有一些其它诸如声音、流畅动画、一些小的交互逻辑...

    但由于还是早期【Dev版本】，很多东西还是不够完善。像圆角设计只有少部分软件支持，微信就不是圆角；老版本的资源管理器会突然出现；开始菜单的固定图标使用起来也不是很方便；任务栏图标居中的时候左边会留白并不是很美观，不知道微软后续如何处理...

    可以期待一下后面【安卓应用支持】功能，感觉会对安卓平板市场造成冲击

    我只能说微软yyds！！！
